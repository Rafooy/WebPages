Brief Description of HTML Elements

DOCTYPE_html

In the beginning of html, we need to define what type of document it is. In our case it is html, therefore it will be <!DOCTYPE html>.

<!DOCTYPE_html>

html

Right after DOCTYPE html, we create the html structure using <html>. At then end of your html structure it should have </html>.

<html> </html>

head

The head element contains the metadata for the webpage and defines the information about the document.Starts with <head> and ends with </head>.

<head> </head>

title

Shows the text you put in head into tab.

<title> </title>

body

The flesh of html where you add what you want to display in your website.

<body> </body>

h1 to h6

The h1 element will display the largest heading text. As heading gets closer to h6, the text will display a smaller heading.

<h1> </h1>

<h2> </h2>

<h3> </h3>

<h4> </h4>

<h5> </h5>

<h6> </h6>

p

Paragraph. Typing a new <p> will create a new paragraph.

<p> </p>

br

The br element is line break. It will create a new line.

<br />

ul

ul stands for unordered list tag and it help create a list of items with no order.

<ul> </ul>

ol

ol stands for ordered list and creates a list with order.

<ol> </ol>

li

li stands for list item tag. It helps ul or ol to list out items.

<ul> <li> </li> </ul>

<ol> <li> </li> </ol>

div

div helps organize html by dividing html into sections. This is useful for css when you want to pick a certain section of your website to cascade. divs are before the section you want to select and right after the ending of the section you want to select.

<div> </div>

span

span is used to group inline-elements in a document. It can also be used to visually change text, but it need to be manipulated by css, javascript, or using style in html. Ex. <span style="color:blue">.

<span> </span>

<!-- -->

<!-- --> is used for comments. It doesn't affect your html and will not display. It is used to leave notes for yourself or others who read your code.

<!-- comments -->

id

id helps specify a unique id for HTML. id is mostly used for CSS and javaScript. The target link is contains # and the element's id.

HTML:

<div id="id_name">

<h1 id="id_name">

<p id="id_name">

CSS:
#id_name{
something:something;
}

class

class helps define attributes with the same class name. To select class in CSS use period before class name.

HTML:

<div class="class_name>

<h1 class="class_name">

<p class="class_name>

CSS:
.class_name {
something : something
}

em

Anything in em will displayed as italacized.

<em>text</em>

strong

Anything in stong will be displayed as bold.

<strong> text </strong>

Images

img src="" allows pictures to be added into html.

<img src="link to picture">

img src="" alt=""

The alt in img src will show a description of the image. Height and width can also be used with img src.

<img src="link to picture" alt="text">

<img src="link to picture" alt="text width="int" height="int">

Videos

Video src allows video to be displayed on html. Controls allows basic function for the video such as play and pause.

<video src="link to video" width="int" height="int controls>

Linking

a allows linking to other website possible. It can also be used to target a specific place on your website.

<a href="link"> text or picture </a>

To a specific part of website:
<a href='specific part of the website. Ex. targetting id with #name'> text or picture </a>

Nav

Nav allows navigation to other websites or on the same page. Intended for major block of navigation links, such as menus.

<nav> </nav>

Table

First step in creating a table, you must use the table element.

<table></table>

Table Border

Table border allows the table to have a border equal to the integer you put in. The integer represents the thickness of the border.

<table border="int"> </table>

Tr

Tr allows the creation of rows in your table. Tr are between table element.

<tr> </tr>

Table Head

Table head allows the creation of heading in tables. The scope in th will allow you to specify if the head is for row or columns. It must be between tr.

<th scope="row">name </th>
<th scope="col> name </th>

Table Data

Td allows the table to receive information of what you want to put in the table and display it. It must be between tr.

<tr> <td>data </td> </tr>

TD_COLSPAN

Td_colspan allows the expansion of a column.

<td colspan="int"> </td>

TD_ROWSPAN

Td_rowspan allows the expansion of row.

<td rowspan="int"> </td>

Tbody

Tbody tag is used for the body of the table. Used after thead and bfore tfoot. tr and td goes into tbody.

<tbody> </tbody>

Thead

Thead tag is used for the beginning of the table. tr abd th usually go in thead tag.

<thead> </thead>

Tfoot

Tfoot is used at the end of the table. tr and td is in tfoot.

<tfoot> </tfoot>
